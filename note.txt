Init Module Golang:
1. Create Folder and CD to Folder
2. Run: go mod init <module-name>

Build: go build
run: go run <file.go>

Function: di go lang nama function harus unik dan tidak boleh duplikat dalam satu tempat yang sama.
Variable: di go variable tidak bisa di overide dengan type yang berbeda. variable harus digunakan ketika di diklarasikan.
Constant Variable: variable yang tidak bisa di ganti nilainya.
Type Decalation: membuat alias untuk type data tertentu
Type Data Array: di go panjang array itu fix tidak bisa dikurangi atau ditambah
Type Slice: potongan dari data array, slice mirip array tapi panjang data nya bisa berubah (dynamic)
